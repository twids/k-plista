# Security Policy

## Supported Versions

| Version   | Supported          |
| --------- | ------------------ |
| 0.0.x     | :white_check_mark: |

## Reporting a Vulnerability

If you believe you have found a security vulnerability in this project, please report it via [GitHub Security Advisories](https://github.com/twids/k-plista/security/advisories) or email the repository owner.

**Acknowledgment:** We will acknowledge receipt of your report within 2 business days.

**Process Timeline:** We aim to provide an initial assessment within 7 business days. You will receive updates at least every 7 days until the issue is resolved or closed.

**Triage and Resolution:** After triage, we will work with you to understand the impact and severity. If accepted, we will coordinate a fix and disclosure timeline. If declined, we will provide a clear explanation.

Thank you for helping us keep our project secure!
